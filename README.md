# BacklogZero Prototype

BacklogZero is a front-end prototype for a gaming backlog website.
It helps users organize games, filter by preferences, discover picks, and explore a mock community space.

## Features

- Home dashboard with quick stats and featured games
- Backlog page with filter panel and game cards
- Quick Pick flow for personalized recommendations
- Discover and Community sections
- Mock user profile page
- Responsive layout for desktop and mobile screens

## Tech Stack

- HTML
- CSS
- Vanilla JavaScript

## Run Locally

1. Open the project folder.
2. Open `index.html` in your browser.

## Notes

- This is a prototype and uses mock data.
- Game artwork is loaded from external URLs.
